# FNAF-like Mini Game

Lightweight Five Nights-style browser game demo.

Files:

- [index.html](fsd-projects/fnaf/index.html) — main game page
- [style.css](fsd-projects/fnaf/style.css) — styles
- [game.js](fsd-projects/fnaf/game.js) — game logic

Open [index.html](fsd-projects/fnaf/index.html) in a browser to play.

Controls:

- Click camera buttons to view rooms.
- Toggle `Light` and `Door` to defend against animatronics (uses power).

This is a simple demo and intended as a starting point.

Adding real assets

- Place sound files in `fsd-projects/fnaf/assets/` named `ambient.mp3`, `click.mp3`, and `jumpscare.mp3` to use them instead of the built-in synth.
- Place an image `jumpscare.png` in the same folder to use as the visual jumpscare.
